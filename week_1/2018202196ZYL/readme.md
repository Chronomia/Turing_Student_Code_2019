代码基本思路（最终版）：
先将信院首页wget下来并放入set和queue中，然后执行BFS函数，在宽度优先搜索的框架下爬下queue队列末尾的第一个网页，然后利用正则表达式去匹配每一行的url，过滤掉非网页的url，然后再将有效的url扩展成http开头的完整形式，最后将该url在set之中查重，不重复的放入queue中，循环至queue中元素被清空。
